Introducción

Copia y pega texto, imagenes y ficheros entre los equipos conectados en la misma red con dos sencillos pasos.

Con las teclas Ctrl + Alt + 1 (Mac Osx: Command + Shift + 1) para compartir los datos seleccionados por la ventana de la aplicación en los distintos equipos.

Cuando querámos pegar esa información compartida pulsaremos Ctrl + Alt + 2 (Mac Osx: Command + Shift + 2).
